Preguntas frecuentes

De tener dudas acerca de nuestros servicios ofrecidos le ofrecemos una lista de preguntas frecuentes

  • Al adquirir uno de los paquetes web, ¿El nombre del dominio es para siempre?

    El nombre de dominio es adquirido por un año (o más tiempo, esto a convenir), luego del final del tiempo de compra se debe renovar anualmente.

  • En los paquetes web se menciona sobre un máximo de MB de almacenamiento, que pasa si mi informacion es mayor al paquete que voy a adquirir?

    Tendrá acceso a la cantidad de MB adquirido mediante su paquete web, de necesitar mas debe comunicarse con nosotros previamente, el tope de su servicio aquirido es automático, el servidor le indicará un error al tratar de sobrepasar el límite.

  • Mi página web ocupa la mitad de los MB de mi servicio adquirido, ¿puedo subir mis fotos y documentos personales?

    Recuerde que es un servidor exclusivo web, el espacio de los correos (de todas las cuentas email) ocupan cierto espacio en el servidor dependiendo de la cantidad de correo que tenga, de estar lleno el espacio asignado ya no podrá recibir correos electrónicos. Se le aconseja solo usar el servicio para transacciones web o caso contrario puede contactarnos para brindarle una solución

  • ¿Qué significa CMS?

    CMS, Content Management System, cuya traducción es: Sistema de Gestión de contenidos. Una plataforma web que nos permite personalizar y modificar una web, esta plataforma funciona mediante plantillas, son de fácil manejo para el usuario e intuitivo.

  • ¿Tengo que tener la página web abierta en mi computador todo el tiempo para que me escriban y yo poder responder?

    No necesariamente. Es configurable e incluso puede llegarle las notificaciones al teléfono movil mediante una aplicación de terceros. Puede designar varios usuarios que esten involucrados en las notificaciones de atención.
